### 安装依赖

`npm install`

### 启动

`npm run watch`

### 执行 `npm link`

此时就可以使用 `lucky` 命令了。

- `lucky init vue-template myVue`
- `lucky config get`
- `lucky config set type orgs`
- `lucky config set registry vuejs-templates`

- `lucky config set type users`
- `lucky config set registry YvetteLau`

### 发布

开发完成后，即可发布至 npm.
